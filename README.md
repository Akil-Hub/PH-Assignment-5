    - 1️⃣ What is the difference between var, let, and const?

    Answer 1: var ,let and const all are javascript variable declaration method. 

    VAR: if we declare any variable with var we can re assign it with the same name and new value. and var is hoisted like global variable.

    LET: if we declare any variable with let we can
    change this value but can't re assign  the same value name with new value. we use let when the value of the variable needs to updated in DOM for any reason.

    CONST: Const means Constant variable. if we declare any variable with const we never re assign its value and can't change it.
